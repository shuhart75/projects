# Прототип

Каталог для материалов по прототипу диалога с AI-агентом на странице симуляции.

## Что хранить здесь

- сценарии пользовательских потоков;
- наброски интерфейса и тексты сообщений агента;
- результаты проверок и замечания.

## Минимальный набор артефактов

- `user-flow.md` — шаги пользовательского сценария;
- `dialog-scripts.md` — примерные реплики и ветвления;
- `validation-notes.md` — выводы по тестам прототипа.

## Текущий интерактивный макет

- `simulation-agent-mock.html` — кликабельный макет вкладки симуляции с интеграцией AI-агента по требованиям.
